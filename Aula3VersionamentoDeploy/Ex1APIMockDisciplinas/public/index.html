Mini API de Disciplinas para uso nas aulas de PSW1 e PSW2

Método Rota Descrição
GET /time/todos Rota get para obter todos os times
GET /time/nome/:nome Rota get para obter um time pelo nome
POST /time/inserir Rota post para inserir um novo time
PUT /time/atualizar Rota put para atualizar um time, pelo nome
DELETE /time/excluir Rota delete para excluir um time, pelo nome
